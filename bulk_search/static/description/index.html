Overview

The Advance Bulk, Quick Search module is designed to enhance the search functionality within the Odoo application.
It introduces features such as separate search, bulk search, and quick bulk related search, allowing users to efficiently find relevant records across various models.

Install Module "Advance Bulk/Quick Search" . Once Module Installed, You Can View Search Bar

Configuration

Go to the Settings menu, Look for 'Configure Bulk Search' in settings.



  • Create records to perform an advanced search.Make sure that when creating records, the model has a 'Name' field.
  • Example: I have added Models like "Sale Order, Invoices, MRP, Products" To perform In search Box.

Separate Search

  • The separate search feature can perform from anywhere within the application.
  • Enter the search query in the provided search box.
  • Example: If I enter "S000" and click inside the search box or hit "ALT+SHIFT+E", a dropdown will appear with related records containing with Model Namw with Scroll Facility










  • Select a record from the dropdown , by Selecting from Dropdown Automatically fills that Name in Search Box

  • Click the search icon button to navigate to the appropriate view of that record.

  • "We can navigate or access our advanced search wherever needed."







Bulk Search With Comma separated

  • Navigate to the search box.
  • Enter the search query in the search box, for example: P00008,P00007,P00006.
  • Use only commas (",") for bulk search.




  • Click on the search icon or press Enter to initiate the bulk search.
  • Results will be displayed based on the entered query.
